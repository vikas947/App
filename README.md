# Multi-Purpose Financial Calculator

A premium Flutter fintech-style mobile app featuring EMI, Investment, Property, Currency, and Age calculators.

## Highlights
- Clean modular structure (`screens/`, `widgets/`, `providers/`, `models/`, `theme/`)
- Light / Dark / System theme with persisted preferences
- Riverpod state management
- fl_chart-powered visualizations
- Currency API with SharedPreferences cache fallback

## Run
```bash
flutter pub get
flutter run
```
